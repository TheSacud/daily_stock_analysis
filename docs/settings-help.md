# Settings Help

Settings help provides short, field-specific guidance in the Web settings page.

## Content Requirements

Each setting should explain:

- What the setting controls.
- When to change it.
- Expected value format.
- Runtime impact.
- Relevant caveats or examples.

## Maintenance

When adding or changing settings, update the registry, `.env.example`, settings help content, and related docs together. New settings should work without configuration when possible and should enhance behavior when configured.
