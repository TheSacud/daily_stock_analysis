# Intelligence Sources

Intelligence sources provide news, announcements, catalysts, risks, and context used by analysis and reports.

## Principles

- Prefer fresh, source-attributed information.
- Separate hard information such as filings and announcements from softer news or commentary.
- Keep provider diagnostics visible when data is missing, stale, or rate-limited.
- Do not let one failed source block the entire analysis unless that source is explicitly required.

## Typical Sources

The system can use search providers, news APIs, exchange or data-provider announcements, and provider-specific company context. Availability depends on configured API keys and market coverage.

## Output Expectations

Intelligence output should include positive catalysts, negative risks, neutral context, source freshness, and uncertainty. Summaries should avoid overstating unverified rumors.
